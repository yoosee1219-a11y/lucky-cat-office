## 🚀 토스 콘솔 등록 단계별 가이드

### Step 1: 토스 개발자 콘솔 접속
1. https://console.toss.im 접속
2. 토스 계정으로 로그인

### Step 2: 새 앱 등록
1. "앱 만들기" 클릭
2. 앱 ID 입력: `lucky-cat-office`
3. 앱 이름 입력: `운빨냥사원`

### Step 3: 기본 정보 입력
- 영문명: Lucky Cat Office
- 카테고리: 전략
- 웹뷰 URL: https://lucky-cat-office.vercel.app

### Step 4: 디자인 자료 업로드
- 앱 아이콘: public/app-icon.png
- 정방형 썸네일: public/thumbnail-square-1000.png
- 가로형 썸네일: public/thumbnail-wide-1932.png

### Step 5: 앱 설명 입력
- 한 줄 요약 복사 붙여넣기
- 상세 설명 복사 붙여넣기
- 키워드 입력

### Step 6: 게임 등급 선택
- 자체 등급 분류 선택
- 등급: 전체 이용가
- 근거 입력 (위 내용 참고)

### Step 7: 제출 및 검토 대기
- 모든 정보 확인 후 제출
- 토스 검토 대기 (보통 1-3일)
